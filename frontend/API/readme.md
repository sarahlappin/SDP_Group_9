## How to run API test
1. Download the following files:
    - `api-eieio.py`
    - `heatmaps.py`
    - `main.js`
    - `test.html`
2. Run the following commands
    ```
    $   pip install flask
    $   pip install bson
    $   pip install flask_pymongo
    $   pip install -U flask-cors
    ```
3. Then run `python api-eieio.py`
4. Open `test.html` in your browser
